Contexte

Les maladies cardiovasculaires (MCV) sont responsables de millions de décès chaque année. À travers l'étude Framingham, nous explorons les données démographiques, les habitudes de vie et les indicateurs médicaux pour comprendre les facteurs qui augmentent le risque de maladies cardiovasculaires dans les 10 ans. Cette analyse met en lumière les relations entre ces facteurs et permet de dégager des profils à risque.

Questions avec Visualisations Interactives

  • Quels groupes démographiques sont les plus exposés aux maladies cardiovasculaires (MCV) ?
  • Quels facteurs médicaux sont les plus déterminants dans le risque ?
  • Comment les habitudes de vie comme le tabagisme ou l'exercice influencent-elles ce risque ?

Prédictions Cardiaques Basées sur le Machine Learning

PulseML intègre un modèle de Machine Learning avancé pour prédire le risque de maladies cardiovasculaires en se basant sur des données telles que l'âge, la pression artérielle, le cholestérol et d'autres facteurs de santé. Le modèle analyse les corrélations entre ces variables et applique des algorithmes prédictifs, comme la régression logistique ou les arbres de décision, pour estimer la probabilité d'apparition de maladies. Cette fonctionnalité permet de fournir des prédictions précises et exploitables, aidant les utilisateurs à identifier les facteurs de risque critiques et à prendre des décisions préventives éclairées.

Compréhension des Facteurs de Risque

PulseML vise à éclairer les utilisateurs sur les facteurs de risque majeurs des maladies cardiovasculaires. En visualisant les données sous forme de graphiques interactifs, l'application permet d'identifier les relations clés entre les indicateurs médicaux, les habitudes de vie, et les risques potentiels. Cette approche aide à mieux comprendre comment des variables comme l'âge, le cholestérol ou le tabagisme contribuent au développement des maladies cardiaques, offrant ainsi une meilleure prise de conscience collective sur les enjeux de santé publique.
