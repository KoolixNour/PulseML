Chapitre 1 - Détails

Bienvenue dans le chapitre 1. Voici deux graphiques basés sur les données de Framingham :

Taux de Cholestérol Moyen par Âge
Description du graphique

Ce graphique montre comment le taux de cholestérol moyen varie avec l'âge. On observe généralement une augmentation du taux de cholestérol avec l'âge, atteignant un pic dans les tranches d'âge entre 50 et 60 ans. Cette tendance peut être liée à des changements dans le métabolisme et au mode de vie.

Nombre de Fumeurs par Catégorie d'Éducation
Description du graphique

Ce graphique illustre la corrélation entre le niveau d'éducation et le nombre de fumeurs. Les données montrent que les individus avec un niveau d'éducation plus faible ont une prévalence plus élevée de tabagisme. Cela peut être attribué à un manque d'accès à l'information sur les dangers du tabac.
